# 我的项目

这是一个使用 React 构建的简单展示页面。

## 可用的脚本

在项目目录中，你可以运行：

### `npm start`

在开发模式下运行应用。
打开 [http://localhost:3000](http://localhost:3000) 在浏览器中查看。

### `npm run build`

将应用构建到 `build` 文件夹。

### `npm run deploy`

将应用部署到 GitHub Pages。
